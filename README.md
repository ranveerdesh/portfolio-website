# Portfolio Website

A modern, responsive portfolio website built with Next.js, React, and Tailwind CSS.

## Features

- **Two-Section Layout**: Fixed left section with profile info and navigation, scrollable right section with content
- **Responsive Design**: Fully responsive design that works on all devices
- **Sections**:
  - Experience: Professional work history
  - Skills: Technical skills organized by category
  - Projects: Showcase of key projects
  - Achievements: Notable accomplishments and awards
- **Dark Mode**: Automatic dark mode support based on system preferences
- **Smooth Scrolling**: Smooth navigation between sections

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
npm start
```

## Technologies Used

- **Next.js 16**: React framework for production
- **React 19**: JavaScript library for building user interfaces
- **TypeScript**: Type-safe JavaScript
- **Tailwind CSS 4**: Utility-first CSS framework
- **Lucide React**: Beautiful icon library

## Customization

Edit `app/page.tsx` to customize the portfolio content with your own:
- Name and title
- Professional summary
- Experience details
- Skills
- Projects
- Achievements
- Social media links
